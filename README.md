# Šifrování

navrhněte funkci _decrypt_content()_, která:
- vstup:
	- `key`: 32 bytový klíč zakódovaný base64 (tak jak ho generuje/akceptuje knihovna Fernet)
    - `encrypted_content64`: zašifrovaná data kódovaná jako base64
    - `out_file`: název výsledného dešifrovaného souboru (binární obsah)
- výstup (vedlejší efekt):
    zapíše do souboru `out_file` obsah dešifrovaný pomocí knihovny Fernet
